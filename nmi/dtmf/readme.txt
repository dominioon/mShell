DTMF - mShell NMI module to send DTMF (Dual Tone Multi Frequency) tones during
       an active call.
-------------------------------------------------------------------------------
The module provides the following functionality:

* dtmf.send(tones) -> null 
    Sends tones as DTMF tones. The input can contain only numbers 0-9, * and #.
